What is Full Stack?

Full Stack Web Developer

Full stack web developer is a person who can develop both client and server software.



In addition to masterin HTML and CSS, he/she also knows how to:



Program a browser (like using Javascript, jQuery, Angular, or Vue)

Program a server (e.g. using PHP, ASP, Python, or Node)

Program a database (e.g. using SQL, SQLite, or MongoDB)
