Data Types: 2ye ayrılır 1- Primitive: Tek bir değer tutan türlerdir. Null Undefined Boolean Number : Negatif-pozitif ondalıklı sayıları kapsar. Çok büyük sayılar Big-Int(iger) olarak yazılıyor. String // Numbers: (primitive) // Strings: (primitive) let x; // Now x is undefined let length = 16; let color = "Yellow"; x = 5; // Now x is a Number let weight = 7.5; let lastName = "Johnson"; x = "John"; // Now x is a String // Booleans (primitive) let x = true; let y = false; 2- Non-Primitive (Object) Type: Referansa bağlı. Object Array // Object: (non-primitive) const person = {firstName:"John", lastName:"Doe"}; // Array object: (non-primitiv) const cars = ["Saab", "Volvo", "BMW"]; // Date object: (non-primitive) const date = new Date("2022-03-25");
